## Installation

Add this line to your application's Gemfile:

    gem 'eventick_api'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install eventick_api

create *eventick_api.rb* in <rails_app_folder>/config/initializers 

	require 'eventick'

## Usage

**Getting Token**

	auth = Eventick::Auth.new
	
	auth.email = email
	auth.password = password

	auth.token
